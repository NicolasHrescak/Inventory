Ambiente de execução: https://www.onlinegdb.com/

Para compilar o projeto:
1 - Abra o link https://www.onlinegdb.com/
2 - Faça o upload do arquivo main.hs dentro da IDE.
3 - Selecione a linguagem de compilação Haskell dentro da IDE.
4 - Execute apertando em "Run".
